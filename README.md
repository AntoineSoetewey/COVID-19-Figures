# Covid-19-plots

Made by [Antoine Soetewey](https://www.antoinesoetewey.com/), [Niko Speybroeck](https://twitter.com/NikoSpeybroeck) and [Angel Rosas-Aguirre](https://twitter.com/arosas_aguirre).
